# react-native-anim-splash-screen

Add Lottie animation to splash screen

## Installation

```sh
npm install react-native-anim-splash-screen
```

## Usage

```js
import { multiply } from 'react-native-anim-splash-screen';

// ...

const result = await multiply(3, 7);
```
